Created and implemented an interactive Loan Analytics Dashboard to analyze bank loan performance and credit risk across multiple segments. The project focused on transforming raw loan data into actionable insights that help management monitor default rates, profitability, and lending patterns by age, gender, and loan type.
Tools & Skills: Excel (Power Query, Pivot Tables, Charts, Slicers, Dashboard Design, Data Visualization), Financial Data Analysis, Credit Risk Assessment, KPI Design, Business Insights.
